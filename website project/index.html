Smart application interface design

Flex Box
A flexible box model that we use to design the interface of applications and websites.
CSS Grid
CSS-Grid is a two-dimensional system (as opposed to flexbox, which is one-dimensional), which means that it allows you to configure both rows and columns (in flexbox, our div could contain a row or a column, but never both).

Document Object Model

In addition to the fact that the browser loads and displays the content of the HTML document using CSS style sheets, it also remembers the MODEL of a given document, i.e. it remembers tags, their attributes, order, and hierarchy. The DOM treats each tag, attribute, and text as a separate object, called a node.

Object Oriented Programming

A way of creating software that uses objects - data merging elements (usually called: attributes) and functions (usually called methods). The program consists of objects that communicate with each other and perform various tasks.
